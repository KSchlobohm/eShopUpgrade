## Key Architecture Decisions

- **No big-bang rewrite:** Every milestone is independently buildable
- **Topological order:** Libraries first, web project last (minimizes coupling)
- **BinaryFormatter:** Must be replaced (hard blocker for .NET 9+)
- **Autofac:** Replaced with built-in DI (simpler than upgrading Autofac)
- **log4net:** Optional replacement with Microsoft.Extensions.Logging
- **systemweb-adapters:** Available for transitional compatibility but not recommended long-term
